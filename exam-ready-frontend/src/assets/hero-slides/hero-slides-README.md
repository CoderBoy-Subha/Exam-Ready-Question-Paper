# Hero slide images

Drop any number of screenshots here (`.png`, `.jpg`, `.jpeg`, `.webp`) — the
home page carousel picks them up automatically on the next build/reload.
No code changes needed.

**Naming controls order and caption:**

    01-upload-your-material.png   -> caption "Upload your material"
    02-configure-the-paper.png    -> caption "Configure the paper"
    03-download-your-paper.png    -> caption "Download your paper"

- The leading number controls slide order (natural sort — `2` sorts before `10`).
- Everything after it becomes the caption: dashes/underscores become spaces,
  first letters capitalized.
- A "Start Generating" call-to-action slide is always appended automatically
  as the last slide — don't add one yourself.

Removing a file removes its slide. That's it.
