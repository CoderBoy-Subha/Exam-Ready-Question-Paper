## 📌 Project Overview
**Exam-Ready** is a full-stack web application that uses **IBM Watsonx.ai** to instantly generate balanced question papers from uploaded syllabus materials. It is designed for:
- **Teachers** – Create exam papers in under 10 minutes (95% faster than manual work).
- **Students** – Access unlimited free mock tests with instant scorecards.

---

## 🚀 Key Features
- AI-powered question generation (MCQs, Short Answer, Long Answer)
- Customizable difficulty ratio (Easy/Medium/Hard)
- Syllabus upload (PDF, DOC, TXT)
- Student mock test portal with timer
- Instant scorecard with correct answers
- Export question papers as PDF/ZIP
- Cloud storage on IBM Cloud Object Storage
